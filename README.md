# Favorite Recipes Project

This is the first project in my **Odin Web Development Journey**.

It's a simple HTML project that showcases recipes for my all-time favorite foods. Initially, the project was built using just HTML. As the lessons progressed, I enhanced it by applying **CSS styling** to improve the layout and appearance.

---

## Skills Required

### HTML Skills
- Understanding the basic structure of HTML
- Using elements and tags
- Formatting text content
- Inserting images into the page
- Creating ordered and unordered lists
- Writing proper commit messages

### CSS Skills
- Adding styles to HTML with CSS
- Using class and ID attributes for targeting elements
- Styling specific elements with various CSS selectors
- Understanding the cascade, specificity, and inheritance in CSS
- Using the element inspector to:
  - Access and navigate the DOM
  - Select and inspect specific elements
  - Test and tweak HTML/CSS in real-time
- Working with the **box model** (margin, padding, border, content)
- Differentiating between `block` and `inline` elements
- Understanding the concept of **normal flow**
- Recognizing and using `<div>` and `<span>` elements correctly
